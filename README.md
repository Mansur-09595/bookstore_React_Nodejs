## Запуск проекта

Клонировать проект:

```bash
$ git clone https://github.com/Mansur-09595/bookstore_React_Nodejs
```

Перейти в директорию проекта и установить зависимости

```bash
$ cd bookstore_React_Nodejs && npm install
```

Запустить приложение

```bash
 npm run start
```
